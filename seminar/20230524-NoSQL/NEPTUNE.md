# Neptune

## Graph Data Model

* Property Graph : Gremlin, openCypher (by neo4j)
* Resource Description Framework : RDF, SPARQL

**Property Graph**

* vertex(node) : Data + Property
* edge(relation) : (from, to) + Property
* label : tags of vertex

**RDF**

Tuple
* Subject : vertex. Data
* Predicate : edge
* Object : ??
